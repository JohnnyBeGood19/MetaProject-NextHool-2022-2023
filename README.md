# NextHool Landing Page Prototype

A prototype landing page developed as part of the **Metaproject** training course.

## About Metaproject

Metaproject is an integrated training course that combines:
- **Design Thinking** – User-centered problem-solving approach
- **Prototyping Techniques** – Creating tangible representations of ideas
- **Programming** – Technical implementation and development
- **Computer Processing** – Advanced computational skills

The course takes place in a collaborative and safe learning environment, encouraging students to develop cross-disciplinary skills and adopt a creative, multidisciplinary approach to problem-solving.

## About This Project

This prototype showcases a multidisciplinary approach to web design and development, reflecting the core principles of the Metaproject course: creativity, cross-functional collaboration, and problem-solving through iterative design and technical implementation.

## Technologies

- **HTML** – Structure and semantic markup
- **CSS** – Styling and responsive design
